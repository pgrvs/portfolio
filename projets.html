Inscription - Judo de Vesoul

Description

Ce site permet aux encadrants du Judo de Vesoul d'inscrire chaque adhérent dans Dolibarr grâce à son API. Ainsi, l'adhérent reçoit par email sa facture et les documents à retourner tels que l'attestation de santé ou un certificat médical.

Cela permet au club de Judo de regrouper toutes les informations au même endroit et de facturer les licences de judo selon la catégorie.

Outils utilisés

Framework :

  Next.js

Langage :

  Javascript

Bibliothèque JS :

  React.js

Accueil Client chez Eurydice

Description

Site permettant à l'entreprise Eurydice d'enregistrer un nouveau client sans passer par le site Doliabrr*, grâce à l'utilisation d'une API proposé par Doliarr.

Il est possible d'accéder à tous les clients de l'entreprise ainsi que de créer un diagnostic c'est-à-dire une analyse du matériel informatique client pour trouver la solution au problème rencontré par un client.


* Dolibarr : c'est est un progiciel de gestion intégré et gestion de la relation client open source pour les entreprises.

Outils utilisés

API :

 Utilisation de l'API Dolibarr

Langage :

  PHP

  Javascript

  Twig

API Quiz et Jeu Quiz

Description

Ce site permet aux utilisateurs de jouer à un quiz.
L'utilisateur peut choisir la catégorie du quiz ainsi que le nombre de questions.

Ce site est implémenté en Javascript, qui consomme un API Symfony. Cette API REST propose deux endpoints un pour récupérer les catégories et l'autre pour les questions et réponses en fonction d'un nombre de questions demandé.

Outils utilisés

Framework :

  Symfony pour l'API

Langage :

  JavaScript
