Accueil Client chez Eurydice

Stage de 1ère année de BTS Informatique

Sujet du projet

Ce projet avait pour objectif de fournir à l’entreprise Eurydice un outil permettant d’enregistrer de nouveaux clients sans passer par l’interface classique de Dolibarr, en exploitant une API proposée par ce dernier.

L’application permettait également de consulter la liste complète des clients de l’entreprise. Elle offrait en outre la possibilité de créer un diagnostic détaillé, dans lequel l’utilisateur pouvait décrire le matériel informatique du client ainsi que les problèmes rencontrés. Ces informations étaient ensuite transmises à Dolibarr, où l’entreprise pouvait les traiter et assurer le suivi.

Dolibarr est un logiciel open source de gestion d’entreprise intégrant des fonctionnalités de CRM, ERP et de facturation. Il offre une interface pour gérer les aspects financiers et administratifs.

Outils utilisés

Langages : PHP, Javascript, Twig, HTML, CSS
