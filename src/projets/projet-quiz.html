API Quiz et Jeu Quiz

2e année de BTS Informatique

Sujet du projet

Ce site offre aux utilisateurs la possibilité de participer à un quiz personnalisé. Ils peuvent sélectionner la catégorie de leur choix ainsi que le nombre de questions souhaité.

L’application est développée en JavaScript et s’appuie sur une API REST réalisée avec Symfony. Cette API propose deux endpoints principaux : l’un permettant de récupérer les catégories disponibles, et l’autre fournissant les questions et réponses en fonction du nombre de questions spécifié.

Outils utilisés

  • Frameworks : Symfony, Bootstrap
  • Langages : Javascript, HTML, CSS
