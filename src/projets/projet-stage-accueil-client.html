Accueil Client chez Eurydice

Stage de 1ère année de BTS Informatique

Sujet du projet

Ce projet vise à fournir à l’entreprise Eurydice un outil permettant d’enregistrer de nouveaux clients sans passer par l’interface classique de Dolibarr, en exploitant une API proposée par ce dernier.

L’application permet également d’accéder à la liste complète des clients de l’entreprise. De plus, elle offre la possibilité de créer un diagnostic détaillé, permettant d’analyser le matériel informatique du client afin d’identifier et de résoudre efficacement les problèmes rencontrés.

Dolibarr est un logiciel open source de gestion d’entreprise intégrant des fonctionnalités de CRM, ERP et de facturation. Il offre une interface pour gérer les aspects financiers et administratifs.

Outils utilisés

Langages : PHP, Javascript, Twig, HTML, CSS
